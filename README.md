## Getting Started

How to run Seven locally? 
To setup of Seven for local development and contribution follow the details. 

### Prerequisites

Required
* [python 3.7 above] (https://www.python.org/)
* [npm and node] (https://docs.npmjs.com/cli/v8/commands/npm-install)
* [git] (https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/SupTechRM/Seven
   ```
3. Install Packages
   ```sh
   pip install -r requirements.txt
   ```
4. Setup Frontend
   ```js
   $ cd app
   $ npm install --save-dev electron
   ```
